## Getting Started:

- Clone and install dependencies:

```bash
mkdir ~/Dev/dictionary -p
cd ~/Dev/dictionary
git clone https://github.com/Arvind-4/dictionary.git .
npm install
```

- Run development server:

```bash
npm run dev
```

Open [localhost:5173](http://localhost:5173/) in your Preferred Browser.
